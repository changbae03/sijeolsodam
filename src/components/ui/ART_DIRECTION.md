# 시절소담 — 이미지 아트 디렉션 (v2)

> 모든 사진이 같은 프리미엄 라이프스타일 매거진에서 나온 것처럼 느껴져야 합니다.
> 식료품몰이 아니라 잡지.

## 핵심 원칙

1. **부드러운 자연광** — 정오의 강한 직사광 X, 흐린 날 창가 같은 부드러운 빛
2. **따뜻하고 차분한 색보정** — 무드 있게 살짝 톤다운, 과채도 X
3. **미니멀한 에디토리얼 스타일링** — 소품 과하게 늘어놓지 않기
4. **차분한 구도, 일관된 비율** — 매번 같은 느낌의 프레이밍
5. **여유 있는 네거티브 스페이스 + 옅은 그림자** — 사진이 답답하게 꽉 차지 않도록

## 🥔 식재료 사진

- **원물만** — 조리된 음식 절대 금지
- 배경: **아이보리 리넨 / 라이트 우드 / 뉴트럴 스톤** 중 하나
- 산만한 구도, 스톡포토 느낌 금지
- 카드에 쓸 때는 이미지를 화면 끝까지 채우지 않고 **여백을 두고 옅은 그림자로 살짝 띄운 느낌**을 줌
  (`.ingredient-frame` 클래스 참고)

## 🍳 레시피 사진

- **완성된 요리만** — 원물 사진 금지
- 식재료 사진과 **같은 보정 톤·조명**을 유지 (더 비비드하게 가지 않기)
- 플레이팅은 단순하고 우아하게 — 소품/장식 과하지 않게

## ⚠️ 절대 섞지 않기

같은 콘텐츠 타입 안에서 원물 사진과 완성 요리 사진을 섞지 않습니다.

## 기술 규칙

| 항목 | 규칙 |
|---|---|
| 비율 | 식재료 카드: `5:4`(큰 카드)/`1:1`(작은 카드) · 레시피 카드: `4:5` 고정 |
| 보정 | 모든 이미지에 `.img-editorial` 클래스 적용 |
| 식재료 카드 프레이밍 | `.ingredient-frame` — 아이보리 배경 위에 여백+옅은 그림자로 띄움 |
| 레시피 카드 프레이밍 | 풀블리드(꽉 채움) + 하단 그라디언트 오버레이, 식욕을 돋우는 임팩트 우선 |
| 모서리 | `16px`~`20px` rounded |

## 공통 색보정 필터

```css
.img-editorial {
  filter: saturate(0.92) contrast(0.96) brightness(1.04) sepia(0.05);
}
```

## 식재료 카드 프레이밍 (신규)

원물 사진이 "여유 있는 네거티브 스페이스 + 옅은 그림자"로 보이도록,
아이보리 리넨톤 배경 위에 이미지를 살짝 띄우는 패딩 프레임을 적용합니다.

```css
.ingredient-frame {
  background: var(--color-cream-warm); /* 아이보리 리넨 톤 */
  padding: 14px;
}
.ingredient-frame > img,
.ingredient-frame > span {
  border-radius: 14px;
  box-shadow: 0 6px 20px -4px rgba(44, 42, 38, 0.12);
}
```

## 적용된 컴포넌트

- `HeroIngredientCard` (홈 — 오늘의 제철) — `.ingredient-frame` 적용
- `/shop` 식재료 카드 — `.ingredient-frame` 적용
- `RecipeCard` (전역) — 풀블리드 유지, `.img-editorial`만 적용
- `/ingredient/[name]` 상세 hero — 풀블리드 유지 (몰입형 큰 이미지이므로 예외)
