## 핵심 원칙

1. **부드러운 자연광** — 정오의 강한 직사광 X, 흐린 날 창가 같은 부드러운 빛
2. **따뜻한 중성톤** — 베이지, 아이보리, 우드, 리넨. 차가운 톤(파랑/시안 캐스트) 지양
3. **미니멀한 배경** — 시각적 잡음 없이, 한 가지 소재에 시선이 집중되도록
4. **차분한 구도** — 너무 역동적이거나 기울어진 앵글 지양, 안정적인 중앙/3분할 구도
5. **과채도 지양** — 비비드하거나 채도가 과한 색감 X, 스톡포토 느낌 X

## 콘텐츠 타입별 규칙

### 🥔 식재료 카드 (오늘의 제철, 제철 달력, 장보기)
- **원물 사진만** — 조리된 음식 절대 금지
- 카드 구성: 이미지 → **하단에 "제철 N~M월" 배지 + 한 줄 설명**
- 배경: 우드보드, 리넨, 바스켓 등 따뜻한 소재 위에 놓인 원물

### 🍳 레시피 카드 (오늘의 메뉴 추천, 레시피 목록, 추천 레시피)
- **완성된 요리 사진만** — 원물 사진 금지
- 카드 구성: 이미지 → **조리시간 + 난이도만 간결하게** (그 외 텍스트 최소화)
- 배경/색감: 식재료 카드와 동일한 따뜻한 보정 톤 유지 (음식 사진이라고 더 비비드하게 가지 않기)

### ⚠️ 절대 섞지 않기
같은 콘텐츠 타입(예: 레시피 카드 그리드) 안에서 원물 사진과 완성 요리 사진을 섞어서 보여주지 않습니다.

## 기술적 통일 규칙

| 항목 | 규칙 |
|---|---|
| 비율 | 식재료: `5:4`(큰 카드) / `1:1`(작은 카드) · 레시피: `4:5` 고정 |
| 크롭 | `object-cover`, 중앙 또는 약간 위쪽 기준 |
| 보정 | 모든 이미지에 `.img-editorial` 클래스 적용 (공통 색보정 필터) |
| 모서리 | `16px`~`20px` rounded |
| 오버레이 | 텍스트가 사진 위에 올라가는 경우만 하단 그라디언트(`ink/70 → transparent`) |

## 공통 보정 필터

`globals.css`의 `.img-editorial` 클래스를 모든 식재료/레시피 이미지에 적용합니다.
원본 사진의 출처가 달라도(Unsplash 등 여러 소스) 동일한 톤으로 보이게 만드는 보정값입니다.

```css
.img-editorial {
  filter: saturate(0.92) contrast(0.96) brightness(1.04) sepia(0.05);
}
```

- `saturate(0.92)` — 과채도 방지
- `contrast(0.96)` — 너무 강한 명암 대비 완화 (차분한 톤)
- `brightness(1.04)` — 살짝 밝게, 자연광 느낌
- `sepia(0.05)` — 미세하게 따뜻한 색 캐스트 추가

## 적용된 컴포넌트

- `FeaturedIngredientCard` (홈 — 오늘의 제철)
- `RecipeCard` (전역 — 오늘의 메뉴 추천, 레시피 목록, 추천 레시피)
- `/ingredient/[name]` 상세 페이지 hero 이미지
- `/shop` 식재료 카드
